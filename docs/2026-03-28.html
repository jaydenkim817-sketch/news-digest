Sony temporarily suspends memory card sales due to shortages 악재

Sony가 메모리 카드 공급 부족으로 일시적 판매 중단, 반도체 공급망 차질 발생.

주요 기술 기업의 부품 공급 차질은 산업 전반의 생산 차질과 수급 불균형을 초래할 수 있음.

번역 요약 · The Verge

소니가 2026년 3월 27일부터 CFexpress와 SD 메모리 카드의 거의 모든 제품에 대한 주문을 중단한다고 발표했다. CFexpress Type A, Type B 및 SDXC/SDHC 카드가 영향을 받으며, 일부 Type B 모델과 저가형 SF-UZ 시리즈 SD 카드는 계속 생산될 예정이다. 이는 공식 판매처와 일반 소비자 모두로부터의 주문이 당분간 받아지지 않음을 의미하며, 기존 재고가 소진될 때까지 제품을 구매할 수 있을 것으로 보인다.
